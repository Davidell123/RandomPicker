## Project Title
Random Picker 

## Description
O aplicatie mobila interactiva care permite extragerea aleatorie de elemente dintr-o baza de date comuna si ofera functionalitate pentru adaugarea de noi idei in timp real. Arhitectura este pe 3 niveluri: client (Android), server (Python) si stocare (Firebase Cloud).

## Features
- Selectarea unei categorii din interfata grafica
- Extragerea si afisarea unui element aleatoriu de pe server
- Adaugarea de noi optiuni direct in baza de date din cloud

## Technologies Used
- Java
- Android SDK
- Python (Flask) pentru Server
- Firebase Realtime Database
- REST API (OkHttp3)

## How to Run
1. Clone repository
2. Open with Android Studio
3. Run the Python server locally (`python server.py`)
4. Run on emulator or device
